### Flat Flex Properties
* 2 Layers
* Polyimide Flex
* FPC Thickness: 0.10 mm
* Stiffener: BOTTOM Plastic : 0.20 mm , BOTTOM Fr4 : 0 mm , BOTTOM Metal : 0 mm
* Total thickness at stiffener should be 0.30 mm.
* Stiffener location: See image
